Instrucciones para añadir LuckiestGuy-Regular.ttf

1) Descarga la fuente "Luckiest Guy" (archivo LuckiestGuy-Regular.ttf) desde Google Fonts:
   - Abre https://fonts.google.com/specimen/Luckiest+Guy
   - Haz clic en "Download family" y extrae el archivo .ttf

2) Copia `LuckiestGuy-Regular.ttf` dentro de esta carpeta `fonts/`.

3) Al abrir `index.html`, p5.js cargará la fuente automáticamente (se usa `loadFont('fonts/LuckiestGuy-Regular.ttf')`).

Si prefieres descargar por línea de comandos en Windows (PowerShell), coloca el enlace directo al .ttf y ejecuta:

```powershell
Invoke-WebRequest -Uri "https://ejemplo.com/LuckiestGuy-Regular.ttf" -OutFile "fonts\LuckiestGuy-Regular.ttf"
```

Reemplaza la URL por la del archivo .ttf que descargues.
